# Route53 DNS Zone

```yaml
apiVersion: aws.project-planton.org/v1
kind: Route53Zone
metadata:
  name: example.com
spec:
  records:
    - name: test-a.example.com.
      recordType: A
      values:
        - 1.1.1.1
    - name: test-cname.example.com.
      recordType: CNAME
      values:
        - some-other.example.com.
```
