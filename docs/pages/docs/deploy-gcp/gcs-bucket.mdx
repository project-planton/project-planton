# GCS Bucket

```yaml
apiVersion: gcp.project.planton/v1
kind: GcsBucket
metadata:
  name: my-bucket-name
spec:
  gcpProjectId: <gcp-project-id>
  gcpRegion: asia-south1
  isPublic: false
```


## Validation

```yaml
apiVersion: gcp.project.planton/v1
kind: GcsBucket
metadata:
  name: my-bucket-name
spec:
  isPublic: false
```

```shell
> project-planton validate first-gcs-bucket.yaml
validation error:
 - gcp_project_id: value is required [required]
 - gcp_region: value is required [required]
```
